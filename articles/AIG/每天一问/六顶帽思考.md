# 爱德华·德·博诺的“六顶思考帽”模型

爱德华·德·博诺（Edward de Bono）提出的 “六顶思考帽”（Six Thinking Hats）是一种平行思维工具，旨在帮助个人或团队系统化、结构化地全面思考问题，避免传统讨论中常见的争论、情绪干扰和思维混乱。

该模型的核心思想是：将复杂的思考过程分解为六个清晰的角色（帽子），每次只戴一顶帽子，聚焦一种思维模式。这样可以提升决策效率、激发创造力，并促进协作。

🎩 六顶思考帽的含义与作用
帽子颜色   思维类型           关键问题 / 作用                                                                   特点
白色 🧢   事实与信息       “我们有哪些数据？缺少什么信息？什么是已知/未知？”                                   客观、中立、基于事实

红色 ❤️   情感与直觉       “我对这件事的感觉是什么？我的直觉告诉我什么？”                                       情绪化、非理性、无需解释

黑色 ⚫   谨慎与风险       “哪里可能出错？有什么风险？为什么这行不通？”                                        批判性、防御性、逻辑质疑

黄色 💛   乐观与价值       “好处是什么？为什么值得做？潜在机会在哪里？”                                         积极、建设性、关注价值

绿色 🟢   创新与可能性     “有没有新想法？能否换种方式？如何改进？”                                            创造性、发散、突破常规

蓝色 🔵   控制与组织       “我们的目标是什么？下一步该用哪顶帽子？如何总结和决策？”                             元认知、流程管理、主持人角色

🔄 使用方式（典型会议流程）

1. 蓝帽开场：明确议题、目标和议程。
2. 白帽收集：列出所有相关事实、数据、现状。
3. 红帽表达：允许成员说出直觉感受（不需理由）。
4. 黄帽探索：寻找方案的优势和积极面。
5. 黑帽评估：识别潜在问题、障碍和风险。
6. 绿帽创造：提出新点子、替代方案或优化建议。
7. 蓝帽收尾：综合信息，形成结论或行动计划。

✅ 关键原则：  
- 一次只戴一顶帽子 → 避免思维混杂；  
- 所有人同时戴同一顶帽子 → 实现“平行思维”，而非对抗；  
- 黑帽虽重要，但不能滥用 → 防止陷入消极批判。

✅ 应用场景

- 团队决策会议（产品、战略、项目评审）
- 创意头脑风暴
- 风险评估与可行性分析
- 冲突调解
- 个人复杂问题的自我梳理

🌟 优势

- 减少无效争论：批评的是“想法”，不是“人”；
- 全面覆盖视角：强制考虑情感、风险、创新等维度；
- 提升效率：结构化流程避免跑题；
- 包容不同性格：内向者也能在框架下发言；
- 促进协作：从“我对你错”转向“我们一起解决问题”。

📚 经典语录（德·博诺）

“思考不是天赋，而是一种技能。”  
“六顶思考帽不是关于‘谁对谁错’，而是关于‘如何更好地思考’。”
