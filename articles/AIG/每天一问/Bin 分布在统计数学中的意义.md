# **Bin 分布**在统计里的含义，可以概括为：

## 一、Bin 是什么

**Bin（箱）**：把连续变量按取值区间划分成若干段，每一段就是一个 Bin。例如把 0～100 分成 `[0,20)`、`[20,40)`、`[40,60)`、…，每段就是一个 Bin。

## 二、Bin 分布在统计上的意义

| 概念                   | 含义                                              |
| ---------------------- | ------------------------------------------------- |
| **离散化**       | 把连续变量变成离散的区间，便于计数和统计          |
| **频数分布**     | 每个 Bin 内样本数量，即该 Bin 的频数              |
| **直方图基础**   | 横轴是 Bin 边界，纵轴是各 Bin 的频数或占比        |
| **经验分布近似** | 用 Bin 频数近似总体概率分布，是密度估计的一种形式 |

## 三、在 Sizing 校准里的 Bin 分布

在 SST 的 Sizing 模型中，Bin 分布表示：

- **横轴**：输入量 X（如 Intensity、ECD）被 Cut Lines 切成的 7 个区间
- **纵轴**：落在每个 Bin 里的缺陷数量

意义包括：

1. **样本在各区间的分配**：看出小缺陷、大缺陷、特殊类型（Tall、Dim 等）各有多少。
2. **校准可靠性**：某 Bin 内样本太少，该 Bin 的 Scale Factor 估计不稳定；样本多则相对可靠。
3. **分段是否合理**：若某 Bin 几乎空，或大量样本挤在一个 Bin，可能需要调整 Cut Lines。

## 四、数学表达

设 Cut Lines 为 \(c_1 < c_2 < \cdots < c_6\)，则 Bin 定义为：

- Bin 0: \((-\infty, c_1]\)
- Bin 1: \((c_1, c_2]\)
- …
- Bin 6: \((c_6, +\infty)\)

Bin 分布是一组频数：
\[
n_i = \#\{j : X_j \in \text{Bin}_i\}, \quad i = 0,1,\ldots,6
\]

或对应的比例（相对频率）：\(p_i = n_i / n\)，可近似作为 X 落入各 Bin 的“概率”估计。
