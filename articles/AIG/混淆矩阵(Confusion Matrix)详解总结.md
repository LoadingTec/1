# 混淆矩阵（Confusion Matrix）总结

## 一、基本概念
混淆矩阵是用于评估分类模型性能的基本工具，通过统计模型分类结果中的正确和错误数量，以表格形式直观展示。适用于二分类及多分类问题，是ROC曲线和AUC值计算的基础。

## 二、核心指标
### 1. 一级指标（基础统计量）
- **TP（True Positive）**：实际为正例，预测为正例
- **FN（False Negative）**：实际为正例，预测为负例（Type II错误）
- **FP（False Positive）**：实际为负例，预测为正例（Type I错误）
- **TN（True Negative）**：实际为负例，预测为负例

### 2. 二级指标（性能指标）
| 指标名称         | 计算公式                 | 说明                          |
|------------------|--------------------------|-----------------------------|
| 准确率（Accuracy）  | (TP+TN)/(TP+FN+FP+TN)    | 整体预测正确的比例            |
| 精确率（Precision） | TP/(TP+FP)               | 预测为正例中的实际正例比例    |
| 召回率（Recall）   | TP/(TP+FN)               | 实际正例中被正确预测的比例    |
| 特异度（Specificity）| TN/(TN+FP)              | 实际负例中被正确预测的比例    |

### 3. 三级指标
- **F1-Score**：精确率和召回率的调和平均数  
  `F1 = 2*Precision*Recall/(Precision+Recall)`

## 三、应用特点
1. 适用于二分类和多分类问题
2. 可衍生出多个评估指标（如准确率、F1值等）
3. 与ROC曲线、AUC面积共同构成分类模型评估体系
